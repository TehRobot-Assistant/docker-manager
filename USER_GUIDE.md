## Common Tasks

### "My game server is lagging"
Try hitting **Restart** â€” this often clears up performance issues.

### "Server won't start"
If **Start** doesn't work, wait 30 seconds and try again. If it still won't start, contact the admin.

### "I need access to another container"
Contact the admin to have it added to your account.
